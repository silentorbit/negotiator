# Negotiator - Chrome Extension

*The web is not a done deal.*

You no longer need to blindly accept what the web offers.
This extension allows you to set your own terms.

Simple http request and response filtering for advanced users..

# Installation

Install from the [Chrome Web Store: Negotiator](https://chromewebstore.google.com/detail/negotiator/lfopjlendebbnfddpgpoaahmpbgmffii)

# Try unpacked

- git clone https://github.com/silentorbit/negotiator.git
- Go to chrome://extensions/
- Enable developer mode
- Click the button "Load unpacked"
- Choose the "extension" folder in the cloned git repo
